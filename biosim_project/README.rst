======
Biosim
======

This is a project that simulates the ecosystem of rossumøya. 
There are five different landscape types: Jungle, Savannah, Desert, Mountain and Ocean.
There are two kinds of animals living on the island: Herbivores and Carnivores.


Contents
--------

- biosim: Python package for simulation of the ecosystem of Rossumøya
- examples: examples of how the simulation works and run, with visualization.
